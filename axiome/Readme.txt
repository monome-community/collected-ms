#	Infos:

#	* Using Monomeserial.
#	* Designed for the Monome 40h.
#	* Made with Pd-0.40.3-extended.
#	  on an iBook G4 800
#
--------------------------------------------
#	Features:
#
#	* 7 tracks Mlr-style loop sequencing.
#	* Pattern play
#
--------------------------------------------
#	Quick Howto:
#
#	* Set monome prefix to /box
#	* Set monome ports to 8000 & 8081
#	* Load samples.
#	* Tweak the sample parameters
#	  (slice amount and slice length).
#	* The rest is pretty straightforward.
#	* Eventually save your song (sample
#	  locations along with parameters) and
#	  call it back later.
#
#	* See Man.txt for more infos
#
